# git-practice
Soft Eng X practice season 
Objective
- Learn the basic concepts of Git and GitHub.
- Create and initialize a local Git repository.
- Track project files using Git commands.
- Commit changes with appropriate messages.
- Push the project to a GitHub repository.

 Outcome

- Successfully created a local Git repository.
- Added and committed project files.
- Connected the local repository to GitHub.
- Pushed the project to the remote repository.
- Gained hands-on experience with the basic Git workflow.
- Learned how to manage project versions using Git.
